Campos de Formularios de HTML Interactivos

type="text"
Campo de texto de una sola línea. Se usa para nombres, títulos, ciudades, etc.
Ejemplo: Ingresar tu nombre completo.
type="password"
Oculta los caracteres ingresados. Ideal para contraseñas y datos sensibles.
Ejemplo: Crear una contraseña segura.
type="email"
Valida que el formato sea de correo electrónico. Útil para registros y contactos.
Ejemplo: Ingresar tu email para recibir notificaciones.
type="number"
Permite solo números. Se usa para cantidades, edades, precios, etc.
Ejemplo: Indicar tu edad.
type="tel"
Campo para números telefónicos. Muestra teclado numérico en móviles.
Ejemplo: Ingresar tu número de teléfono.
type="url"
Valida que el texto sea una dirección web válida. Útil para portafolios, redes, etc.
Ejemplo: Compartir tu página web.
type="date"
Permite seleccionar una fecha desde un calendario. Útil para cumpleaños, reservas, etc.
Ejemplo: Seleccionar tu fecha de nacimiento.
type="time"
Permite seleccionar una hora. Útil para agendar reuniones o recordatorios.
Ejemplo: Indicar la hora de tu cita.
type="color"
Permite elegir un color usando un selector visual. Útil para personalización.
Ejemplo: Seleccionar tu color favorito.
type="range"
Permite seleccionar un valor dentro de un rango definido. Útil para encuestas.
Ejemplo: Califica tu experiencia del 0 al 10.
type="file"
Permite seleccionar y subir archivos desde el dispositivo. Útil para imágenes, documentos, etc.
Ejemplo: Adjuntar una foto de perfil.
type="radio"
Permite seleccionar solo una opción entre varias. Útil para elegir género, opciones únicas, etc.
Ejemplo: Seleccionar tu género.
type="checkbox"
Permite seleccionar o deseleccionar una opción. Útil para aceptar condiciones, suscripciones, etc.
Ejemplo: Marcar si aceptas los términos y condiciones.
type="hidden"
No se muestra al usuario, pero se envía con el formulario. Útil para IDs, tokens, etc.
Ejemplo: El usuario no ve este campo, pero el servidor sí lo recibe.
type="search"
Campo optimizado para búsquedas. Puede mostrar una X para limpiar.
Ejemplo: Buscar un producto en una tienda.
type="datetime-local"
Permite seleccionar fecha y hora (sin zona horaria). Útil para eventos, reservas, etc.
Ejemplo: Indicar cuándo será el evento.
type="month"
Permite seleccionar un mes y año. Útil para periodos escolares, facturación, etc.
Ejemplo: Seleccionar el mes de inicio de clases.
type="week"
Permite seleccionar una semana específica del año. Útil para planificaciones.
Ejemplo: Indicar la semana de entrega de un proyecto.
<textarea>
Campo de texto de varias líneas. Útil para comentarios, descripciones largas, etc.
Ejemplo: Dejar una sugerencia o comentario.
<select>
Permite seleccionar una opción de una lista desplegable. Útil para países, ciudades, categorías, etc.
Ejemplo: Elegir tu país de residencia.
type="button"
Botón que no envía el formulario. Se usa para acciones personalizadas con JavaScript.
Ejemplo: Mostrar un mensaje de ayuda.
type="submit"
Envía el formulario al servidor. Útil para finalizar el registro, compra, etc.
Ejemplo: Enviar los datos ingresados.
type="reset"
Restablece todos los campos del formulario a su valor inicial. Útil para limpiar datos.
Ejemplo: Borrar todos los campos y empezar de nuevo.
